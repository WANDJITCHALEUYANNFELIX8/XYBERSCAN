🎭 Rôles et Responsabilités
Nous accueillons tous les profils:


Développeurs Python : Développement des modules du scanner (Niveau : Bases de Python).


Testeurs QA : Tests du scanner sur des environnements réels (Aucun prérequis technique).


Rédacteurs : Documentation et rapports en français.


Designers : Conception de l'interface visuelle future (Figma/Canva).

⚙️ Workflow GitHub
Pour contribuer, nous utilisons le GitHub Flow:


Forkez le dépôt principal vers votre propre compte GitHub.

Créez une branche pour votre fonctionnalité : feature/nom-de-la-fonctionnalite.

Effectuez vos modifications et faites des commits clairs et descriptifs.

Poussez vos modifications sur votre fork.

Ouvrez une Pull Request (PR) vers la branche principale du dépôt Xyberclan.

Toute PR doit être approuvée par au moins un membre Core avant d'être fusionnée.

💻 Standards de Code
Pour maintenir un code propre et lisible, veuillez respecter ces règles :


Style : Suivez les conventions PEP 8.


Documentation : Chaque fonction doit être commentée en Français.



Langue : Le code lui-même (variables, fonctions) s'écrit en Anglais.


Bugs : Utilisez les "Issues" GitHub pour signaler tout problème ou suggestion.

⚖️ Éthique et Légalité
En contribuant à ce projet, vous vous engagez à respecter les valeurs de Xyberclan:

Le scanner est destiné à un usage éducatif et défensif uniquement.

Il est strictement interdit de scanner des sites sans autorisation explicite du propriétaire.
